This project implements a basic Electronic Voting Machine using Arduino.
It allows users to cast votes for different candidates using push buttons.

## Features
- Multiple candidate voting buttons
- Vote counting using Arduino
- Reset functionality
- Result display using Serial Monitor / LEDs

## Components Used
- Arduino Uno
- Push Buttons
- Breadboard
- Resistors
- Connecting Wires

## Applications
- Educational demonstration of voting systems
- Digital electronics and embedded systems learning
